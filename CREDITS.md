# TrueTune — Third-party credits

Phase 0 dependencies:

- **Oboe** (https://github.com/google/oboe) — Apache 2.0. Low-latency audio I/O.
- **AndroidX / Jetpack Compose / Material 3 / Hilt** — Apache 2.0.
- **Kotlin** — Apache 2.0.

No audio samples are bundled. Phase 3's metronome click and Phase 4's drone
are synthesized in DSP, so there is no sample-library licensing to declare.
